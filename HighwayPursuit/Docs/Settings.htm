Settings

From the main menu you may select any of the following settings and options detailed below. The main menu can be reached from within the game by pressing Escape.

To navigate through the menus, use the Cursor Up and Down keys to select an item, Cursor Left and Right to cycle through a list of options for the selected item (where present) and Enter to perform the action specified for an item (where appropriate).

New game
Start a new game of Highway Pursuit. Any existing game that is in progress will be lost.
High scores
View the current high score table.
Options
Configure the graphics, sound and controller options (see below).
About Highway Pursuit
Displays the full credits list for the game.
Quit to Windows
Quit Highway Pursuit and return to Windows. Any existing game that is in progress will be lost.

Graphics Options

Selecting Options and then Graphics options from the main menu will allow you to configure the graphics settings for the game. These are as follows:

Resolution
Sets the screen resolution for the game. Higher resolutions will show a much crisper picture, but may cause the game to slow down.
Filtering
Switches filtering on and off. When switched on, the graphics will have a much smoother appearance at the possible expense of performance.
Anti-aliasing
Switches anti-aliasing on and off. When switched on, the edges of objects will have a much smoother appearance at the possible expense of performance.
Brightness
Allows the brightness level of the game to be increased if the game appears too dark.
Vertex calculation
Controls whether the 3d graphics are calculated by the processor (software) or the graphics card(hardware). Normally this should be set to hardware but it can be changed if problems occur.

Sound Options

Selecting Options and then Sound options from the main menu will allow you to configure the sound settings for the game. These are as follows:

Sound Effects
Sets the volume for the game sound effects.
Music
Sets the title screen, in-game and high score music volume.
Speech
Sets the volume for Ashley's radio communications.
Subtitles
Switches subtitles for Ashley's radio communications on and off. Note that if you have both this and Speech switched to off, you will be missing important information from your HQ while you are playing the game.

Controller Options

Selecting Options and then Controller options from the main menu will allow you to configure the controls for the game. These are as follows:

Controller
Allows the keyboard or a compatible joystick, gamepad or steering wheel to be selected. If no items are displayed here other than the keyboard, Highway Pursuit did not recognise any other compatible control devices.
Controller sensitivity
Allows the sensitivity of the selected controller to be set. If the car is not cornering quickly enough, increase the sensitivity; if it is turning too sharply, decrease it.
Dead zone
Sets the size of the "dead zone" when using a joystick/pad or steering wheel. This defines how far away from centre the controller must be moved before the car starts to steer. With the dead zone off, even the slightest move will cause your car to steer; a larger dead zone will need a larger movement of the stick/wheel before the car starts to steer.
Define controls
Allows the keyboard and controller buttons to be configured to your liking. A separate set of controls are maintained for both keyboard and joystick, so you can configure the controls however you like for each controller.
Use joystick for menus
If set to Yes and the Controller is set to a joystick/gamepad/wheel, you will be able to use this controller to navigate the menus as well as play the game. Use the controls that are configured for up and down to move through the menus, left and right to change selections and fire to select menu options.

High score Options

Selecting Options and then High score options from the main menu will allow you to configure the high score settings. These are as follows:

Upload high scores
Highway Pursuit is written to support the Retrospec on-line high score table, and will allow each score to be added to the table. You can either add your scores by entering a code into the Retrospec web site, or you can get Highway Pursuit to upload the scores automatically.

Setting this option to "after every game" will instruct Highway Pursuit to connect to the high score server immediately after you have entered your name into the game's own high score table. This is recommended for when you are playing with a permanent internet connection.

Setting this option to "only from main menu" will instruct the game to only attempt to upload the high scores when you select "Upload high scores" from the main menu. This is recommended for dial-up internet connections.

For more information on the Retrospec on-line high score tables, visit the Retrospec web site.

Clear high score table
This will erase all entries from your local high score table. Ensure you have uploaded any scores to the internet before doing this, as once cleared these scores cannot be recovered.
